# 日报摘要模板

## 文件头（仅新建文件时写入）

```markdown
# 📅 {YYYY}年{MM}月{DD}日 工作日报

> 自动生成于 {生成时间} | 由 AI Session Journal 驱动
```

## 每段对话摘要模板

```markdown
---

### 🕐 {开始时间} ~ {结束时间} | 第 {N} 次对话

#### 📋 需求概述
{一句话描述本轮对话的核心目标}

#### 🎯 关键决策
| 决策 | 原因 |
|------|------|
| {决策1} | {为什么这样做} |
| {决策2} | {为什么这样做} |

#### 📝 对话摘要
{3-5 句话概括整个对话过程，重点写做了什么、解决了什么问题}

#### 📦 产物清单
- `新建` `{文件路径}` — {文件作用}
- `修改` `{文件路径}` — {改了什么}
- `删除` `{文件路径}` — {为什么删除}

#### ⚠️ 待办 / 遗留
- [ ] {待办项1}
- [ ] {待办项2}

#### 💡 经验沉淀
{如果有值得记住的经验、新学到的技术、踩过的坑，写在这里}
```

## 追加规则

当日已有内容时，在文件末尾追加，段间用 `---` 分隔。

追加格式示例：

```markdown
---

### 🕐 14:30 ~ 15:15 | 第 2 次对话

...
```
